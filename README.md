🚀 PriceFinder AI | موتور هوشمند مقایسه قیمت و جستجوی تصویری

معرفی پروژه

PriceFinder AI یک سامانه هوشمند برای مقایسه قیمت محصولات، جستجوی تصویری، ردیابی تغییرات قیمت و یافتن بهترین فروشگاه است.

هدف این پروژه، ساخت یک پلتفرم حرفه‌ای برای بازار ایران است که کاربران بتوانند به‌راحتی محصولات را جستجو، قیمت فروشگاه‌های مختلف را مقایسه و بهترین گزینه را انتخاب کنند.

---

امکانات

نسخه اول (MVP)

- ثبت‌نام و ورود کاربران
- پنل مدیریت
- مدیریت محصولات
- مدیریت برندها
- مدیریت دسته‌بندی‌ها
- مدیریت فروشگاه‌ها
- مقایسه قیمت
- جستجوی محصولات
- API اختصاصی

---

نسخه دوم

- تاریخچه تغییر قیمت
- نمودار قیمت
- علاقه‌مندی‌ها
- نظرات کاربران
- اعلان کاهش قیمت
- پنل فروشندگان

نسخه سوم

- جستجوی محصول با عکس (هوش مصنوعی)

- خزنده خودکار قیمت فروشگاه‌ها
- پیشنهاد هوشمند محصولات مشابه
- داشبورد تحلیل بازار
